## RWPY
RWPY是一个专为游戏Rusted Warfare设计的Python模块，用于游戏相关的Mod开发和地图制作等。
本模块在Python3.10下编写
## 警告
目前本模块尚未完成，不能确保其可用性。
## 安装
第一步：
切换到你想要放置模块的位置，输入：
git clone git@github.com:zerodegress/rwpylib.git
复制模块所在目录绝对路径。
第二步：
切换到你的Python安装位置下，找到Lib/site-packages/文件夹，在其下新建文件“mypath.pth”，内容为你刚才复制的绝对路径。
第三步：
依次输入
python
>>>import rwpy
若未出现报错，说明模块已成功安装。
